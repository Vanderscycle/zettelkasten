**** Ran out of virtual disk space

[] Modify disk within VM
#+begin_src bash
  sudo parted <vm-disk> # e.g. /dev/sda
  # while in parted
  (parted) unit GB
  (parted) resizepart 1 128GB
  # now resize the sub partition
  sudo resize2fs /dev/sda1
#+end_src
