* SystemD
Considered the antichrist by some, but a benefit to all. It runs as ~PID 1~ allows for management of all process (background/foreground) for the linux kernel.

** Systemctl
The cli for it
#+begin_src bash
# can also stop/start/restart/etc/
systemctl -u status emacs
# list the logs
journalctl -u home-manager-henri.service
#+end_src
