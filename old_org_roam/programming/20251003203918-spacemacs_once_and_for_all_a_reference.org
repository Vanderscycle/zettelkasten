**** üß© Basic Text Object Motions
These are the most common and work beautifully with `ys`.

| Text Object | Description                              | Example                  |
|-------------+------------------------------------------+--------------------------|
| `iw`        | inner word (no space)                    | `ysiw)` ‚Üí (word)         |
| `aw`        | *a* word (includes space)                  | `ysaw"` ‚Üí `"word "`      |
| `is`        | inner sentence                           | `ysis)` ‚Üí (Hello world.) |
| `as`        | a sentence (includes space after period) | `ysas)`                  |
| `ip`        | inner paragraph                          | `ysip}`                  |
| `ap`        | a paragraph (includes trailing newline)  | `ysap}`                  |
| `i)` / `a)` | inner / outer parentheses                | `ysi)` or `ysa)`         |
| `i"` / `a"` | inner / outer quotes                     | `ysi"` or `ysa"`         |

**** üß© More advanced motions
| Command    | Meaning                          |
|------------+----------------------------------|
| `ys$)`     | Surround *to end of line*          |
| `ys}}`     | Surround *to next paragraph*       |
| `ys0"`     | Surround *to beginning of line*    |
| `ysf<x>)`  | Surround *from cursor to next `x`* |
| `ys/word)` | Surround *until next ‚Äúword‚Äù match* |
| `ysgss<combo>)` | Surround *until next match* |

*** files ~SPC f~
    ~SPC f e d~ open spacemacs config
    ~SPC f e R~ Reload config

    ~SPC f d~ delete file
    ~SPC f D~ delete file NOW!
    ~SPC f R~ rename file
    ~SPC f c~ save buffer/selection in another file
    ~SPC f b~ bookmarks (alternative ~C-x r b~)
    ~SPC f f~ find files (alternative ~SPC ~a d~) note if you select . you will launch dired
    ~SPC f j~ dired jump (jump opens dired quicker)
    ~SPC f l~ open file literaly (fundemental mode)
      You have asked to visit it literally,
      meaning no coding system decoding, format conversion, or local variables.

    ~SPC f y <>~ yank/copy
      b buffer name
      n file name (usually same as buffer)
      N file name without extention
      d directory path the file is nested in (absolute)
      y file path (absolute)
      Y file path (relative)
    ~SPC f t~ filetree (not recommended)

*** text ~SPC x~
    ~SPC x .~ line movement transient state
    ~SPC x U~ Uppercase everything
    ~SPC x u~ downcase everything
    ~SPC x x~ execute the selection in your language of choice
